# Программа распознавания речи

Локально выполняемое настольное приложение для распознавания речи с GPU-ускорением для Windows. Преобразует разговорную речь в текст, идентифицирует отдельных говорящих, при необходимости переводит результат и предоставляет вывод в нескольких настраиваемых форматах.

**Версия 1.0** — выпущена 21.05.2026

## Возможности

- Преобразование речи в текст через faster-whisper (CUDA-ускорение; поддерживается режим CPU)
- Диаризация говорящих через pyannote.audio — до 10 одновременных говорящих
- Локальный перевод (Helsinki-NLP OPUS-MT, офлайн) или онлайн (Google Переводчик)
- Форматы вывода: обычный текст, DOCX, субтитры SRT, JSON, буфер обмена
- Пакетная обработка файлов с очередью FIFO
- Библиотека голосовых профилей: сохранение и идентификация именованных говорящих в сессиях
- Управление группами говорящих для контекстно-зависимой идентификации
- Настраиваемые глобальные горячие клавиши (системные, работают даже в свёрнутом состоянии)
- Обычный режим записи и режим быстрой сессии
- Режим GUI (CustomTkinter) и полноценный режим CLI
- Интеграция с системным треем с уведомлениями
- История сессий с ретроактивной регенерацией вывода
- Резервное копирование и восстановление всех пользовательских данных

## Системные требования

| Требование | Подробности |
|------------|-------------|
| ОС | Windows 10 или 11, 64-бит |
| GPU | Рекомендуется NVIDIA GPU с CUDA (RTX 3060 Ti или лучше); поддерживается режим CPU |
| ОЗУ | Минимум 4 ГБ; рекомендуется 8 ГБ для файлов длиной от 30 минут |
| Диск | 10 ГБ свободного места (включает все размеры моделей Whisper и языковые пары OPUS-MT) |
| VLC | Требуется для воспроизведения аудио — установщик загружает VLC автоматически при отсутствии |
| FFmpeg | Требуется для извлечения MP4/AVI — должен находиться в PATH при запуске из исходного кода |

## Быстрый старт — Установщик

1. Скачайте `wsp_setup.exe` из [последнего релиза на GitHub](../../releases/latest).
2. Запустите установщик и следуйте мастеру:
   - Выберите модель Whisper (рекомендуется Medium).
   - Примите лицензию HuggingFace pyannote, если требуется идентификация говорящих.
3. Запустите **Программу распознавания речи** с рабочего стола или из меню «Пуск».
4. Выберите устройство ввода или перетащите аудиофайл в очередь пакетной обработки и нажмите **Запуск**.

## Быстрый старт — Из исходного кода

```bat
git clone https://github.com/Leofaidev/SpeechRecognition.git
cd SpeechRecognition
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src/main.py
```

> **Примечание:** PyAudio требует Microsoft C++ Build Tools и PortAudio через vcpkg перед `pip install`.
> Пошаговые инструкции см. в комментариях в начале файла `requirements.txt`.
> FFmpeg должен быть установлен системно и добавлен в PATH.

## Использование командной строки

```bat
wsp.exe --input interview.mp3 --output-format txt json
wsp.exe --input a.mp3 b.wav --output-folder C:\Output --speaker-group "Interviewers"
wsp.exe --backup C:\Backups\wsp_backup.zip
wsp.exe --help
```

Полный справочник параметров и коды выхода см. в [docs/CLI.md](docs/CLI.md).

## Библиотеки

Все библиотеки бесплатны и с открытым исходным кодом. Закреплённые версии см. в `requirements.txt`.

| Библиотека | Назначение | Лицензия |
|------------|------------|----------|
| faster-whisper | Преобразование речи в текст и определение языка | MIT |
| pyannote.audio | Диаризация говорящих (требуется лицензия HuggingFace) | MIT |
| CustomTkinter | Графический интерфейс пользователя | MIT |
| pyaudio | Захват аудио с микрофона и веб-камеры | MIT |
| OpenCV | Доступ к видеопотоку веб-камеры | Apache 2.0 |
| keyboard | Глобальные горячие клавиши | MIT |
| transformers + sentencepiece | Локальный перевод Helsinki-NLP OPUS-MT | Apache 2.0 |
| python-vlc | Воспроизведение аудио/видео | LGPL 2.1 |
| python-docx | Вывод в формате DOCX | MIT |
| PyInstaller | Упаковка приложения | GPL + исключение для загрузчика |
| Inno Setup | Установщик Windows (отдельный инструмент) | Лицензия Inno Setup |
| PyTorch | Среда выполнения глубокого обучения | BSD 3-Clause |

> **Лицензия HuggingFace:** pyannote.audio требует принятия лицензии на модель HuggingFace.
> Если не принята, идентификация говорящих отключается, и все говорящие обозначаются как Неизвестный.
